# Enviroment Docs
- Cofigure a `.env` file in your local project and add it to `.gitignore`.
- Set the folowing Env Variables:
    - FIREBASE_API_KEY =
    - FIREBASE_AUTH_DOMAIN =
    - FIREBASE_DATABASE_URL =
    - FIREBASE_PROJECT_ID =
    - FIREBASE_STORAGE_BUCKET =
    - FIREBASE_MESSAGING_ID =
    - FIREBASE_APP_ID =

## Run in local
- Run `npm run dev`.
